Welcome to Meal Planner

This app syncs in real time between you and your partner's phones. You only need to set this up once — then share the link.

Step 1 — Firebase (live sync)

Firebase is Google's free database. Sign in with your Google account — no new account needed.

1
Go to console.firebase.google.com and sign in
2
Click Create a project → name it "meal-planner" → click through → Create
3
In the left menu click Firestore Database → Create database → Start in test mode → Enable
4
Click the ⚙️ gear icon → Project settings → scroll to "Your apps" → click the </> web icon → register app → copy the config shown
5
Paste the values from the config into the fields below
apiKey
authDomain
projectId
storageBucket (optional)
messagingSenderId (optional)
appId

Step 2 — Anthropic API key (AI features)

Powers recipe extraction from video links and receipt scanning. Get a key at console.anthropic.com → API keys → Create key. Stored only on this device.

Anthropic API key

Your config and API key are saved only on this device. They are never sent anywhere except directly to Firebase and Anthropic.
